Acerca de

Soy desarrollador de Software. Apasionado por la tecnología. Actualmente me encuentro finalizando la Tecnicatura Universitaria en Programación. Mis conocimientos se enfocan en el desarrollo Back-End, profundizando en JAVA y Spring Boot. En forma académica, a través de capacitaciones y de manera personal he realizado diferentes proyectos utilizando múltiples herramientas , como por ejemplo Php , Python - Django, C# .Net y Java - Spring Boot para el Back-End. Html , Css , Javascript y React para el Front-End. Bases de Datos Relacionales , de las cuales he utilizado Mysql , SqlServer y Postgresql y No Relacionales como MongoDb. A través de cursos pude explorar nuevas áreas de gran interés como Testing y Análisis de datos. Mi objetivo es continuar ampliando mis conocimientos y habilidades integrando y contribuyendo en equipos de valor.
